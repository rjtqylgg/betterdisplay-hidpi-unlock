## 先说它到底解决了什么

macOS 对外接显示器一直有个"老毛病"：非 Retina 屏接上去，字体发虚、UI 偏小，调缩放又只能在几个固定档位里挑，挑来挑去都不顺眼。Windows 用户早就习惯了 125%、150% 这种自由缩放，Mac 用户却只能干瞪眼。

BetterDisplay 干的事其实很简单——它通过创建虚拟屏镜像的方式，让系统以为你在用一块更高分辨率的屏幕，再以 HiDPI 模式输出。1080p 的显示器能开出 1920×1080@HiDPI，2K 屏能开出原生 Retina 级别的清晰度。我那块 27 寸 1080p 的便宜文书屏，装完之后开启"高解析度"，瞬间像换了一块屏。

它还能做的事远不止 HiDPI。键盘快捷键直接调外接屏亮度音量（不用再摸显示器上的物理按键了）、XDR/HDR 屏幕解锁全亮度档位、DDC 硬件级控制、虚拟屏、画中画、屏幕串流……功能列表长得像一份说明书。
